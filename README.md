# Quizify - Online Quiz System

Quizify is an Online Quiz Management System built using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap. It allows users to attempt quizzes, view results, earn certificates, and track their performance. Admins can manage quizzes, questions, users, and results from a dedicated dashboard.

## Features

### User

* User Registration & Login
* Dashboard with Available Quizzes
* Free and Paid Quizzes
* Timer-Based Quiz Interface
* Question Navigation Palette
* Instant Results
* Quiz History
* Certificates for Eligible Quizzes
* Leaderboard

### Admin

* Admin Dashboard
* Create, Edit, and Delete Quizzes
* Manage Questions
* Manage Users
* View Results
* AI Quiz Generator
* Certificate Management

## Tech Stack

* Frontend: HTML, CSS, JavaScript, Bootstrap
* Backend: PHP
* Database: MySQL
* Server: Apache (XAMPP)

## Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Clone or download this project.
4. Move the project folder to:

   ```text
   C:\xampp\htdocs\
   ```
5. Create a database named:

   ```text
   quiz_system
   ```
6. Import the SQL file into phpMyAdmin.
7. Update database credentials in `config.php`.
8. Open:

   ```text
   http://localhost/Quizify
   ```
